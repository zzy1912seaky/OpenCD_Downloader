# OpenCD_Downloader

A userscript to streamline torrent downloads on the OpenCD web interface.

- No crawler
- No API calls
- No automated background requests
- All actions are based on normal user-initiated clicks

## Installation

This script is distributed via GreasyFork:

https://greasyfork.org/zh-CN/scripts/565446

1. Install a userscript manager such as Tampermonkey
2. Open the link above
3. Click the Install button

## Usage

- Works only on the OpenCD web interface
- A control panel will appear at the bottom-right corner of the page
- You can scan the current page (up to 50 torrents)
- Optional filter for free torrents only
- Supports single or batch download

## Security Notes

This script does not perform any crawling, API calls, or background polling.

It only simulates normal user clicks on existing web elements and does not bypass any site mechanisms.
