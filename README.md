### 一、功能说明  
  
本脚本用于优化 OpenCD 网页端的种子下载体验，主要解决以下问题：  
  
1. 减少逐个点击种子、进入详情页下载所产生的等待时间  
2. 无需人工肉眼识别免费种子与非免费种子  
3. 当下载流量不足时，可选择仅筛选并下载免费种子  
4. 所有操作均基于网页已有功能，不修改、不绕过站点机制  
  
扫描数量与 OpenCD 当前页面显示的种子数量保持一致，最多 50 条。  
  
官方报备许可证明：  
![ScreenShot_2026-02-07_111235_271.png](https://open.cd/attachments/202602/20260207112409c6fcf6956fd3ea50ca0180e136335dcd.png.thumb.jpg)  
  
---  
  
### 二、主要功能  
  
* 扫描当前页面全部种子（最多 50 条）  
* 可选仅筛选免费种子  
* 支持单个下载与批量下载  
* 下载行为基于网页原生点击流程  
* 页面右下角提供脚本组件入口

### 六、使用方法  
  
本脚本仅适用于 OpenCD 网页版。  
  
页面右下角会出现脚本组件展开入口，使用流程如下：  
  
1. 打开 OpenCD 种子列表页面  
2. 点击“扫描”按钮  
3. 可选勾选“仅筛选免费种子”  
4. 选择单个下载或批量下载  
5. 开始下载
